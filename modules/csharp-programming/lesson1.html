💻 Lesson 1: C# Programming Fundamentals

Learn programming concepts essential for Unity game development

1. What is Programming? 🤔

Concept:

Programming is giving instructions to a computer to perform specific tasks. Just like you follow a recipe to cook food, a computer follows your code (instructions) to do something.

Think of it as teaching a robot to do exactly what you want - but you need to be very clear and precise!

Real-World Example:

Making a sandwich:

  • Step 1: Get two slices of bread
  • Step 2: Put butter on one slice
  • Step 3: Add cheese
  • Step 4: Cover with the other slice

Similarly, programming is writing step-by-step instructions for the computer!

Example: Your First Program
// This is a simple program that displays a message
print("Hello, World!")
print("Welcome to programming!")
Output:
Hello, World!
Welcome to programming!
💡 Tip: The line starting with // is a comment. Comments help explain your code to other programmers (or yourself later!). The computer ignores comments.

2. Variables and Data Types 📦

Concept:

Variables are like containers that store information. Think of them as labeled boxes where you can keep different things.

Data Types tell us what kind of information is stored in the variable (numbers, text, true/false, etc.).

Common Data Types:

Data Type Description Example
Integer Whole numbers 5, -3, 0, 100
Float/Decimal Numbers with decimal points 3.14, -0.5, 2.0
String Text (words or sentences) "Hello", "Programming"
Boolean True or False values true, false
Example: Creating Variables
// Integer variable - storing a whole number
age = 25
print("Age:", age)

// String variable - storing text
name = "Alice"
print("Name:", name)

// Float variable - storing a decimal number
temperature = 36.5
print("Temperature:", temperature, "degrees")

// Boolean variable - storing true or false
isStudent = true
print("Is student?", isStudent)
Output:
Age: 25
Name: Alice
Temperature: 36.5 degrees
Is student? true

🔑 Key Points:

  • Variable names should be meaningful (use age instead of x)
  • Text (strings) must be in quotes: "like this"
  • Variable names cannot start with numbers
  • Use meaningful names: studentName is better than n

3. Basic Operations ➕➖✖️➗

Concept:

Operations are actions you perform on data. The most common operations are mathematical calculations and combining text.

Arithmetic Operations:

Operator Operation Example Result
+ Addition 5 + 3 8
- Subtraction 10 - 4 6
* Multiplication 6 * 7 42
/ Division 20 / 4 5
% Remainder (Modulo) 10 % 3 1
Example: Math Operations
// Basic arithmetic
a = 10
b = 3

sum = a + b
print("Sum:", sum)

difference = a - b
print("Difference:", difference)

product = a * b
print("Product:", product)

quotient = a / b
print("Quotient:", quotient)

remainder = a % b
print("Remainder:", remainder)
Output:
Sum: 13
Difference: 7
Product: 30
Quotient: 3.333333
Remainder: 1
Example: Real-World Calculation
// Calculate total price after discount
originalPrice = 100
discount = 20 // 20% off

discountAmount = originalPrice * (discount / 100)
finalPrice = originalPrice - discountAmount

print("Original Price:", originalPrice)
print("Discount:", discount, "%")
print("You Save:", discountAmount)
print("Final Price:", finalPrice)
Output:
Original Price: 100
Discount: 20 %
You Save: 20
Final Price: 80

String Operations:

Example: Combining Text
// Concatenation - joining strings together
firstName = "John"
lastName = "Smith"

fullName = firstName + " " + lastName
print("Full Name:", fullName)

// Combining strings with numbers
age = 25
message = "I am " + str(age) + " years old"
print(message)
Output:
Full Name: John Smith
I am 25 years old

4. Input and Output 💬

Concept:

Input is data that the user provides to the program (like typing your name).

Output is data that the program displays to the user (like showing results on screen).

Think of it like a conversation: you ask questions (input) and get answers (output).

Example: Interactive Program
// Getting input from the user
print("What is your name?")
name = input() // User types their name

print("How old are you?")
age = input() // User types their age

// Displaying output
print("Hello, " + name + "!")
print("You are " + age + " years old.")
Output Example (when user types "Emma" and "20"):
What is your name?
Emma
How old are you?
20
Hello, Emma!
You are 20 years old.
Example: Simple Calculator
// A simple calculator program
print("=== Simple Calculator ===")

print("Enter first number:")
num1 = float(input())

print("Enter second number:")
num2 = float(input())

// Perform calculations
sum = num1 + num2
product = num1 * num2

// Display results
print("Sum:", sum)
print("Product:", product)
Output Example (when user enters 5 and 3):
=== Simple Calculator ===
Enter first number:
5
Enter second number:
3
Sum: 8
Product: 15
💡 Tip: When getting numbers from input, use int(input()) for whole numbers or float(input()) for decimal numbers. This converts the text input into a number you can do math with!

5. Comments and Code Structure 📝

Concept:

Comments are notes in your code that explain what the code does. They help other programmers (and future you!) understand your code better.

Code Structure means organizing your code in a clean, readable way.

Types of Comments:

Example: Using Comments
// This is a single-line comment
// It starts with two slashes

/*
  This is a multi-line comment.
  You can write multiple lines of explanation
  between /* and */
*/

// Calculate area of a rectangle
length = 10 // in meters
width = 5 // in meters
area = length * width

print("Area:", area, "square meters")

🔑 Good Coding Practices:

  • Use meaningful variable names: price instead of p
  • Add comments: Explain complex parts of your code
  • Keep it simple: Break complex problems into smaller steps
  • Use spacing: Add blank lines to separate different sections
  • Be consistent: If you use camelCase, use it everywhere
Example: Well-Structured Program
// ===== Temperature Converter =====
// This program converts Celsius to Fahrenheit

// Get temperature from user
print("Enter temperature in Celsius:")
celsius = float(input())

// Convert to Fahrenheit using the formula: F = (C * 9/5) + 32
fahrenheit = (celsius * 9/5) + 32

// Display the result
print(celsius, "°C is equal to", fahrenheit, "°F")
Output Example (when user enters 25):
Enter temperature in Celsius:
25
25 °C is equal to 77 °F
