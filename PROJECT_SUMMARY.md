## 📦 Package Scripts Available

```bash
# Development
npm start              # Start local server
npm run dev            # Start with live reload

# Quality Checks
npm run validate       # Run all linters
npm run lint:html      # Check HTML
npm run lint:css       # Check CSS
npm run lint:js        # Check JavaScript

# Formatting
npm run format:css     # Format CSS
npm run format:js      # Format JavaScript
npm run format:html    # Format HTML

# Testing
npm run lighthouse     # Run Lighthouse audit

# Deployment
npm run deploy:netlify # Deploy to Netlify
npm run deploy:vercel  # Deploy to Vercel
npm run deploy:surge   # Deploy to Surge
npm run deploy:firebase # Deploy to Firebase
```
